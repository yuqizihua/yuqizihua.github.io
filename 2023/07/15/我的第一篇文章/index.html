搭建个人博客

GitHub Pages + Hexo 搭建个人博客

  1. 准备工作
    注册 GitHub 账号

    安装 Git（版本控制）

    安装 Node.js

  2. 创建GitHub仓库
    仓库名格式：用户名.github.io 如 yourname.github.io
    勾选 “Initialize with a README”
    选择main分支

    注意可以先生成SSH密钥

    1.生成SSH密钥（如果已有可跳过）：ssh-keygen -t ed25519 -C “xxxxx@qq.com“
    2.将公钥添加到GitHub：复制公钥内容 默认路径 ~/.ssh/id_ed25519.pub
    登录GitHub → Settings → SSH and GPG keys → New SSH key，粘贴公钥
    3.测试SSH连接：ssh -T git@github.com 看到 You’ve successfully authenticated 即成功
    4.重新部署： hexo clean && hexo generate && hexo deploy

  3. 本地安装博客框架

    安装Hexo命令行工具

    npm install -g hexo-cli

    初始化博客项目

    hexo init my-blog
    cd my-blog

    安装依赖

    npm install

  4. 配置与写作
    修改 _config.yml：设置标题、作者、主题等

    新建文章：hexo new “我的第一篇文章”

    用 Markdown 编辑文章：位于 source/_posts/ 目录下

  5. 本地预览与调试
    hexo generate # 生成静态页面
    hexo server # 启动本地服务器（访问 http://localhost:4000）

  6. 部署到GitHub
    安装部署插件：npm install hexo-deployer-git –save
    在 _config.yml 中添加：
    deploy:
    type: git
    repo: git@github.com:yourname/yourname.github.io.git # 替换为你的仓库URL
    branch: main
    执行部署：hexo deploy

  7. 访问你的博客
    打开 https://你的用户名.github.io

    如显示默认页面，即部署成功！
